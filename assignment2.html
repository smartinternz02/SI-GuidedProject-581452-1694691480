import seaborn as sns

import matplotlib.pyplot as plt

from pandas.api.types import CategoricalDtype import warnings

# Load the car_crashes dataset

car_crashes = sns.load_dataset("car_crashes")

# Pairplot

warnings.simplefilter(action='ignore', category=FutureWarning) sns.pairplot(car_crashes)

plt.title("Pairplot of Car Crashes Dataset") plt.show()


Inference for Pairplot Visualization:

The pairplot helps us visualize relationships between numerical variables in the car_crashes

dataset. Some key observations:

  • There are positive correlations between variables such as "total" and "speeding."

  • There are negative correlations between variables such as "alcohol" and "speeding."

  • Some variables have right-skewed distributions.

    # Scatterplot

    warnings.simplefilter(action='ignore', category=FutureWarning) sns.scatterplot(x="alcohol", y="total", data=car_crashes) plt.title("Scatterplot of Alcohol vs. Total Crashes") plt.show()



    Inference for Scatterplot Visualization:

    The scatterplot shows a positive correlation between alcohol consumption and the total number of crashes. As alcohol consumption increases, the total number of crashes tends to increase.


    # Boxplot

    warnings.simplefilter(action='ignore', category=FutureWarning) sns.boxplot(x="speeding", y="total", data=car_crashes)

    plt.title("Boxplot of Speeding vs. Total Crashes") plt.show()



    Inference for Boxplot Visualization:

    The boxplot helps us visualize the distribution of total crashes for different levels of speeding. Key observations:

  • It allows us to identify potential outliers.

  • We can see the spread of the data.

    # Histogram

    warnings.simplefilter(action='ignore', category=FutureWarning) sns.histplot(car_crashes["alcohol"], bins=20, kde=True) plt.title("Histogram of Alcohol Consumption")

    plt.show()


    Inference for Histogram Visualization:

    The histogram displays the distribution of alcohol consumption. Key observations:

  • The distribution appears to be right-skewed, indicating that most states have lower alcohol consumption.

    # Countplot

    warnings.simplefilter(action='ignore', category=FutureWarning) sns.countplot(x="ins_premium", data=car_crashes, palette="Set1") plt.title("Countplot of Insurance Premiums") plt.xticks(rotation=45)

    plt.show()


    Inference for Countplot Visualization:

    The countplot displays the frequency of different insurance premium values. Key observations:

  • It helps us see which premium values are more common among the states.

  • Some premium values are more frequent than others.
